#### How to Run the Project

1. Clone the repo
   ```
   $ https://github.com/usama7871/usama-resume-generator
   ```
2. Install the dependencies
   ```
   $ yarn install
   ```
   or
   ```
   $ npm install
   ```
3. Run the project
   ```
   $ yarn dev
   ```
   or
   ```
   $ npm run dev
   ```
4. Visit http://localhost:3000/ on your browser

### Run via Docker
